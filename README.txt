Написать сервис для хранения данных о пользователях.


При разработке необходимо использовать язык Python. Для работы с БД, реализации веб-сервера и т.д. можно использовать
 любые библиотеки.


1. Аутентификация в сервисе происходит с помощью cookie.

2. Администраторы могут видеть все данные пользователей и изменять их.

3. Простые пользователи могут видеть лишь ограниченное число данных обо всех пользователях и редактировать часть своих
 данных.



######################################################################################################################


        RUN.



python3 -m venv venv              создаем виртуальное окружение.

source venv/bin/activate          активация виртуального окружения.

pip install -r requirements.txt   устанавливаем зависимости.

python manage.py migrate          применяем миграции.

python manage.py createsuperuser  создаем суперпользоватея  (у него  admin = True)

python manage.py runserver        запускаем сервер.

http://127.0.0.1:8000/admin/      логинемся суперпользователем.

http://127.0.0.1:8000/admin/users/user/add/   добавляем обычного пользователя (у него  admin = False)

 ctr+c                             останавливаем сервер.

python manage.py loaddata users    загружаем тестовые данные для наглядности.

python manage.py runserver

________________________________________________________________________________________________________________

       ROUTES:

http://127.0.0.1:8000/login/                Аутентификация в сервисе.

http://127.0.0.1:8000/logout/               Выход.

http://127.0.0.1:8000/users/current/        Получение данных о текущем пользователе.

http://127.0.0.1:8000/users/                Постраничное получение кратких данных обо всех пользователях.
                                            (пагинация настраивается в users.service.py)

http://127.0.0.1:8000/users/{pk}            Изменение данных пользователя. admin = True может менять любого пользователя
                                            admin = False может поменять только свои данные.


http://127.0.0.1:8000/private/users/        Постраничное получение кратких данных обо всех пользователях.

http://127.0.0.1:8000/private/users/{pk}    Детальное получение информации о пользователе. (Только admin = True)
